## NHN Cloud — CSAP 인증 현황

- 국내 공공 클라우드 시장 1위 CSP. IaaS 분야 국내 최초 CSAP 인증
- 전국 13개 IDC 운영 (국내 CSP 중 최다)
- DaaS 인증 보유 유일 사업자 → 공공기관 인터넷망 PC 대체(VDI) 프로젝트 독점적 위치
- CSAP 프로모션: 인증심사·서류 준비 전담팀 + 인프라 사용료 500만원 크레딧 지원
- 공공기관 전용 리전 별도 운영 (민간과 논리적 분리)

## CSAP 인증 사업자 분포 (조사 시점)

- IaaS 인증: 9개사
- SaaS 표준 인증: 22개사
- SaaS 간편 인증: 44개사
- DaaS 인증: 1개사 (NHN Cloud)

## CSAP 등급 인정 규칙

- 기존 IaaS 인증 / SaaS 표준 인증 / DaaS 인증 = **중등급**으로 인정
- 기존 SaaS 간편 인증 = **하등급**으로 인정
- 유효기간: IaaS·SaaS 표준·DaaS = 5년, SaaS 간편 = 3년
- 2024-07 간편인증제 도입 (소규모 기업 부담 완화)
- 2024-06 평가 방식 개편: 첫 인증 평가 후, 동일 구성·환경으로 다른 IaaS에 추가 구축 시 서면 평가로 대체 가능
- 2025+ 사후관리 강화: 서면평가 미흡 시 샘플링 현장점검 추가, 미흡 지속 시 매년 현장평가

## 2027년 검증 제도 통합 (중대 변화)

- 2027년 하반기부터 기존 CSAP + 국정원 보안검증 체계 → 국정원 **'신규 검증 제도'로 단일화**
- 공공 시장: 국정원 신규 검증 제도로 단일화
- 민간 영역: 기업 정보보호 관리체계(ISMS)에 **CSAP가 '모듈화' 형태로 통합** — 자율 선택
- 단일 검증체계 시행 전 CSAP 인증 제품은 기존 유효기간(5년) 그대로 인정
- 검증 항목이 클라우드 기술 특성에 맞게 개선 → 보안 수준 강화 + 기업 부담 경감

## ISMS / ISMS-P 인증 체계

- ISMS (정보보호 관리체계): 정보통신망법 §47. 통제항목 **44개**
- ISMS-P (정보보호 및 개인정보보호 관리체계): 정보통신망법 §27조의2 + 개인정보보호법. 통제항목 **65개** (ISMS 44개 + 개인정보 보호 항목)
- 인증기관: KISA. 유효기간: **2년** (CSAP 5년과 다름). 1년차·2년차 사후관리 평가
- ISMS 의무 대상: 정보통신서비스 매출 100억원+ / 일평균 이용자 100만명+ / 매출·세입 1500억원+ 상급종합병원 / 재학생 1만명+ 학교 / ISP·IDC 사업자(매출 무관)
- ISMS-P 의무 대상: 개인정보 파일 보유 + 매출 300억원+ / 일평균 30만명+ / 매출·세입 500억원+ 상급종합병원 / 재학생 5천명+ 학교
- 최초 의무대상 지정 시 다음 해 8월 31일까지 인증 취득 (정보통신망법 §72, 미이행 시 1천만원 이하 과태료)
- 2024-07 간편인증제: 매출 300억 미만 중소기업 등. ISMS 40~44개, ISMS-P 62~65개로 항목 축소, 수수료 ISMS 50%·ISMS-P 40% 감소. 핵심 보안 항목(인증·비밀번호·암호·악성코드·취약점)은 유지

## CSAP ↔ ISMS/ISMS-P 관계

- CSAP: 클라우드 서비스의 기술적 보안 수준 평가
- ISMS/ISMS-P: 조직 차원의 정보보호 관리체계 평가
- 공공 SaaS 납품: CSAP(필수) + ISMS/ISMS-P(대규모 서비스 시 필요) 동시 보유 필요한 경우 많음
- ISMS/ISMS-P 보유 기업은 CSAP 신청 시 기존 보안 관리 체계 문서 활용 가능 → 절차 간소화
- 2027 통합 후: ISMS 인증을 통해 CSAP 관련 요건 충족 가능 구조로 개편 전망

## 오픈스택 / 온프레미스 자체 구축

- 공공·국방 영역 자체 구축 시: 국가정보원 보안적합성 검증, CC(공통평가기준) 인증 등 필요
- 국가정보자원관리원(NIRS) 등 공공클라우드센터 입주 요건 별도
- 자체 구축 vs 매니지드 서비스 트레이드오프: 자체 구축은 데이터·키·물리 위치 완전 통제 가능하나 인증 부담(시간·비용·인력) 큼
- 멀티·하이브리드 클라우드 전략: 단일 CSP 의존 회피, 민감 시스템과 비민감 시스템 분리 배치
