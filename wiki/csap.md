# CSAP — 클라우드 서비스 보안인증

공공·정부기관에 클라우드 서비스를 납품하기 위한 **핵심 관문**. OfficeAgent를 [[nhn-cloud]]에 배포해 정부기관에 납품하려면 CSAP 등급 요건을 설계 단계부터 반영해야 한다.

## 제도 기반

- **법적 근거**: 「클라우드컴퓨팅 발전 및 이용자 보호에 관한 법률」 제23조의2
- **운영 체계**: 정책기관(과학기술정보통신부) → 인증기관([[gov-agencies|KISA]]) → 평가기관(KISA·지정기관) → 공공부문 기술자문([[gov-agencies|국가보안기술연구소]]) → 인증위원회
- **인증 프로세스**: 신청 → 서면 평가 → 현장 평가 → 취약점 점검 → 인증 심사 → 인증서 발급
- **법적 강제성**: 공공기관 핵심 행정 정보 시스템은 사실상 CSAP 인증 서비스만 도입 가능 (행안부 가이드·과기정통부 고시로 간접 강제)

## 등급제 (2023-12 개편)

2023년 12월 국가기관 시스템 중요도 분류 기준에 따라 **상·중·하 3단계** 도입. 2024년 2월 「클라우드컴퓨팅서비스 보안인증에 관한 고시」 일부개정(안) 행정예고.

| 등급 | 적용 대상 | [[network-isolation|망분리]] |
|------|-----------|------------------------------|
| **상** | 국가 중요 정보·민감 개인정보 | 물리적 망분리 + 고급 암호화·실시간 모니터링·강화 접근통제 |
| **중** | 일반 공공 업무 | 논리적 망분리 필수 |
| **하** | 비중요 정보·단순 정보 공유 | 논리적 망분리만으로 취득 가능 |

- 상등급 = 중등급 + 추가 4개 평가항목
- 하등급은 진입 장벽이 낮아 [[foreign-csp|외산 CSP]]가 노리는 진입점

## 서비스 유형별 통제항목

| 유형 | 통제항목 | 비고 |
|------|----------|------|
| IaaS | 14개 분야 116개 | 물리적 보안·인프라 가용성 중점 |
| SaaS 표준등급 | 13개 분야 79개 | |
| SaaS 간편등급 | 11개 분야 31개 | |
| SaaS 상등급 (개정안) | 14개 분야 64개 | |
| SaaS 하등급 (개정안) | 11개 분야 30개 | |
| DaaS | 14개 분야 110개 | 가상 데스크톱(VDI), 인터넷망 PC 대체 |

**통제 분야**: 관리적·물리적·기술적 보호조치 + 공공기관용 추가 보호조치. 구체 항목 예시 — 접근통제, 암호화, 감사 로그, 데이터 복구, 데이터 센터 물리 보안, 재해 대응 계획.

## 등급 인정 규칙 (기존 인증 → 신등급)

- 기존 IaaS 인증 / SaaS 표준 인증 / DaaS 인증 = **중등급**으로 인정
- 기존 SaaS 간편 인증 = **하등급**으로 인정

## 유효기간 · 갱신

- 유효기간: IaaS·SaaS 표준·DaaS = **5년**, SaaS 간편 = **3년**
- 최초 평가 후 1년 경과 시 사후평가, 이후 매년 1회(총 4회)
- 만료 6개월 전부터 갱신 신청 가능
- 2025+ 사후관리 강화: 서면평가 미흡 시 샘플링 현장점검, 미흡 지속 시 매년 현장평가

## 취득 비용 · 기간

- 기간: 6개월 ~ 1년
- 비용: 5천만원 ~ 1억원+ (인증 수수료 + 보안 체계 구축 + 컨설팅 + 인프라 개선)
- [[nhn-cloud|NHN Cloud]]는 CSAP 프로모션으로 인프라 비용 500만원 크레딧 + 전담팀 지원
- 2024-06 평가 개편: 첫 인증 후 동일 구성·환경으로 다른 IaaS 추가 구축 시 서면 평가로 대체 가능

## 국제 대응

미국 FedRAMP HIGH/MOD/LOW ↔ CSAP 상/중/하 (LI-SaaS ↔ 하등급). 직접 상호 인증 체계는 없음.

## OfficeAgent 마이그레이션 설계 시사점

- OfficeAgent가 처리할 정보 등급을 먼저 분류 → 목표 CSAP 등급(하/중/상) 결정
- 목표 등급에 따라 [[network-isolation|망분리]] 방식(물리/논리)이 갈림
- [[migration-checklist]]의 CSAP 항목 참조
- 2027년 제도 변화는 [[policy-trends]] 참조
